# HTML & CSS Tutorials for Year 10 Students

Welcome to an exciting journey into web development! These tutorials are designed to teach HTML and CSS through fun, hands-on projects.

## Tutorial Structure
Each tutorial includes:
- 📝 Clear objectives
- 🎯 Step-by-step instructions
- 💡 Tips and tricks
- 🎨 Creative challenges
- 🔗 Links to additional resources

## Tutorials List

### Beginner Level
1. **Personal Profile Card** - Learn HTML structure and basic CSS styling
2. **Favorite Movies Gallery** - Master layouts and hover effects
3. **School Schedule Table** - Create organized data displays
4. **Digital Recipe Book** - Work with lists and typography

### Intermediate Level
5. **Interactive Navigation Menu** - CSS animations and transitions
6. **Photo Portfolio Grid** - Flexbox and CSS Grid layouts
7. **Weather Dashboard** - Advanced styling and pseudo-elements
8. **Music Player Interface** - Complex layouts and styling

### Advanced Level
9. **Responsive Blog Layout** - Media queries and mobile-first design
10. **Game Score Tracker** - CSS variables and advanced selectors

## Getting Started
1. Create a new folder for each tutorial
2. Follow the step-by-step instructions
3. Experiment with the provided challenges
4. Share your creations with classmates!
