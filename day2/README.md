# Day 2: JavaScript + Creative Coding 🎨

Welcome to Day 2! Yesterday you learned HTML & CSS. Today we're adding **JavaScript** to make things interactive and creative.

## What's Different Today?
- Yesterday: Static pages that look nice
- Today: Pages that **respond** to your clicks, movements, and input

## Environment
You'll use the same setup as Day 1:
- **vscode.dev** (online VS Code) to write code
- **CodePen.io** as an alternative (paste code and see results instantly)
- Everything runs in your Chrome browser — no installs needed!

## Tutorials

### Tutorial 1: Pixel Art Editor 🎨
Build your own pixel art drawing tool. Click on grid squares to paint them — like Minecraft but in 2D!

### Tutorial 2: Gradient Designer Tool 🌈
Create beautiful color gradients, see them update live, and copy the CSS code to use in your own projects.

### Tutorial 3: Interactive Particle Canvas ✨
Create a page where colorful particles follow your mouse cursor — mesmerizing and beautiful!

## How to Run Your Code

### Option A: vscode.dev
1. Go to [vscode.dev](https://vscode.dev)
2. Create your files (`index.html`, `style.css`, `script.js`)
3. Install the "Live Preview" extension
4. Right-click your `index.html` → "Show Preview"

### Option B: CodePen
1. Go to [codepen.io](https://codepen.io)
2. Paste HTML in the HTML panel
3. Paste CSS in the CSS panel
4. Paste JavaScript in the JS panel
5. See results instantly!

## Quick JavaScript Intro
Before we start, here's what you need to know:

```javascript
// Variables store information
let color = "red";
let size = 16;

// Functions do things
function sayHello() {
    alert("Hello!");
}

// Event listeners respond to clicks
button.addEventListener("click", function() {
    // This runs when the button is clicked
});
```

That's it! You'll learn the rest as you build. Let's go! 🚀
