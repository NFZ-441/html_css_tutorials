# Day 3: Build a Platformer Game 🎮

Welcome to Day 3! Today you're building a real platformer game — a character that runs, jumps, collects coins, avoids enemies, and progresses through levels. This is the same core concept behind Mario, Celeste, and Hollow Knight.

## What You'll Build
A side-scrolling platformer with:
- A player character that runs and jumps
- Platforms to land on
- Coins to collect
- Enemies to avoid
- 3 progressively harder levels
- Score tracking and lives
- Win and game-over screens

## Environment
Same setup as Day 1 and 2:
- **vscode.dev** (online VS Code) to write code
- **CodePen.io** as an alternative
- Everything runs in your Chrome browser — no installs needed!

## Tutorial Structure
This is one big project split into steps. Follow them in order:

| Step | What You Build |
|------|---------------|
| Part 1 | Canvas setup + drawing the player |
| Part 2 | Movement + gravity + jumping |
| Part 3 | Platforms + collision detection |
| Part 4 | Coins + scoring |
| Part 5 | Enemies + lives |
| Part 6 | Levels + winning |

Each part adds to the same files. By the end, you'll have a complete game!

## Controls
- **← →** Arrow keys to move
- **Space** or **↑** to jump

Let's build a game! 🚀
